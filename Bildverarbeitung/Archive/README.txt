IMG = Bilder

LineDetection = Filtert ein aufgenommenes Bild so, dass die größte Region (in diesem Fall die Linie) als binäres Bild ausgegeben wird.

PathList = Erstellt einen Pfad aus Punkten (Pixel Koordinaten), welcher die Linie nachzeichnet/beschreibt. 

States = Erstellt um jeden Punkt des Pfades einen State (greift sich ein n*n großes Array um den Punkt heraus), die Liste an Arrays/States wird dann gespeichert und später ausgegeben.



-> in der finalen Version ist, dann alles nochmal verbessert in die Line Klasse ausgelagert worden.
